# Prisddle

**Prisddle** - AI Robotics Quiz Game with Kinetic Typography

An interactive web-based quiz game testing knowledge of AI, robotics, neural networks, and machine learning concepts.

## Features

- 🧠 10 challenging AI/robotics questions
- 📊 Real-time performance tracking
- 🎮 Interactive game mechanics with lives system
- 🏆 Achievement system
- 💡 Educational explanations for each answer
- 🔊 Audio feedback
- 📱 Responsive design
- ⚡ Fast performance optimizations

## Tech Stack

- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **Backend**: Node.js + Express
- **Server**: Nginx (reverse proxy)
- **Process Manager**: PM2
- **Deployment**: Ubuntu 22.04 LTS

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production

```bash
npm run prod
```
